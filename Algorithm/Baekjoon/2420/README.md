# 백준 2420번 - 사파리 월드

## 문제 링크
- [백준 2420번 - 사파리 월드] (https://www.acmicpc.net/problem/2420)

## 문제 요약
- 두 도메인의 유명인 N, M이 주어지면, (-2,000,000,000 ≤ N, M ≤ 2,000,000,000) |N-M|을 출력하는 문제

## 풀이 아이디어
- abs()를 이용하여 절댓값을 계산하여 출력했다. 

## 최종 코드
    # 백준 2420 - 사파리 월드

    N, M = input().split()

    N = int(N)
    M = int(M)

    # 절댓값으로 변환한 값을 출력력
    print(abs(N - M))
